Use passive when the action matters more than who did it, the doer is unknown, or you want a formal / academic tone.

Active vs Passive

Active — subject acts
The chef cooked the meal.
Passive — subject receives
The meal was cooked by the chef.

The formula

Subject  +  to be (conjugated)  +  past participle  +  [by + agent]
to be — changes with tense
V3 — past participle
by + agent — optional

All tenses at a glance

Present simple
is/are + V3
The report is written daily.
Past simple
was/were + V3
The door was opened.
Future simple
will be + V3
The cake will be eaten.
Present perfect
has/have been + V3
It has been fixed.
Past perfect
had been + V3
It had been stolen.
Present continuous
is/are being + V3
It is being repaired.

🔑

Modals are auxiliary verbs expressing ability, possibility, obligation, or permission. They never change form and always precede a bare infinitive.

Core modal verbs

can / could
Ability · Permission · Possibility
She can speak Italian.
Passive: It can be done.
must
Strong obligation · Certainty
You must wear a seatbelt.
Passive: It must be signed.
should
Advice · Expectation
You should rest today.
Passive: It should be checked.
may / might
Possibility · Formal permission
It might rain tomorrow.
Passive: It may be delayed.
will / would
Future · Conditional
I will call you later.
Passive: You will be called.
have to
External obligation
We have to leave now.
Passive: It has to be paid.

Modal + passive formula

Subject  +  modal  +  be  +  past participle
modal stays unchanged
be — always bare infinitive
V3 — past participle

Examples in context

can
This form can be submitted online.
must
All passports must be checked.
should
Errors should be reported immediately.
might
The meeting might be postponed.

Common mistakes

✗ Wrong
It must is done today.  /  It should be did.
✓ Correct
It must be done today.  /  It should be done.
